## 概要

ひたすらオレンジ色の玉を避けるだけのゲームです。

## 操作方法

上下左右キー(カーソルキーといいます)を押すとプレイヤー(赤い箱みたいなもの)が動きます。

## 仕様

スコアが 100000 点になるとクリア判定になります。

リセットしたい場合は F5 キーもしくは更新ボタンを押してください。
